# 🎮 Kiddlr (Kids Learning Platform) – Antigravity System Design

## 🚀 Overview

Kiddlr (formerly Kiddokeys / Kiddotaps) is an interactive learning platform designed for children to improve typing, creativity, and cognitive skills through gamified experiences.

---

## 🧱 1. Core Architecture

### 🌐 Frontend
- Next.js 15 (App Router)
- React 19
- TypeScript

### 🧠 State Management
- Zustand v5

---

## 🎮 2. Interactive Engine

### 🎨 Graphics
- PixiJS v8.6.6

### 🔊 Audio
- Tone.js v15

### ✨ Animations
- Framer Motion

---

## 🔥 3. Backend (Firebase)

- Authentication (Google, Email/Password)
- Cloud Firestore (real-time DB)
- Cloud Storage (artwork)
- Firebase Admin SDK

---

## 🎨 4. UI System

- Tailwind CSS 4
- Lucide React (icons)
- Recharts (analytics)

---

## 🏗 5. Monorepo

- Turborepo

Structure:
apps/
  web/
  admin/

packages/
  ui/
  types/
  utils/

---

## 📡 6. External API

- Datamuse API (words, rhymes, opposites)

---

## 📲 7. PWA

- Next-PWA (offline support, installable)

---

## 🔄 8. System Flow

Child → Game Engine → Zustand → Firestore → Dashboard → Parent

---

## 🧠 9. Antigravity Principles

- Learning through play
- Zero friction UX
- Instant feedback
- Passive tracking

---

## 🛡 10. Safety

- Parent/Child roles
- Secure auth

---

## 🚀 11. Future Scope

- AI-based learning
- Speech recognition
- Multi-language
